# TRPG Studio (シナリオ・マネージャー) 仕様書

## 1. アプリケーション概要
Markdown記法に対応した、TRPG向けのシナリオ作成・管理に特化したエディタアプリケーションです。React および Electron をベースに構築されており、Glassmorphism (リキッドガラス風) の最新モダンUIを採用することで、高度な操作性とデザイン性を両立するよう設計されています。

## 2. システム構成
- **フロントエンド**: React 18, Vite
- **スタイリング**: Tailwind CSS (独自アニメーション、backdrop-blur等の拡張適用)
- **ビルド・実行環境**: Electron (デスクトップアプリケーション化用)
- **アイコン**: lucide-react

## 3. 主要データ構造
シナリオの実体データは以下を含む JSON オブジェクトとして、ローカルの指定ディレクトリに自動保存されます。
- `pages`: ページ情報の配列 (要素は `{ id, title, content }`)
- `characters`: 登場人物の名前リスト (文字列配列)
- `images`: 埋め込み画像データ・マッピング (id 等)

### 保存先・レジリエンス機能
- **ストレージパス**: デフォルトの保存先ディレクトリはユーザーの「設定」画面から自由に変更可能です。
- **データ保護**: 対象ディレクトリ・ドライブが一時的に切断されるなどのI/Oエラー発生時には、自動的にアプリ内部 (`localStorage`) へ退避バックアップを実行し、復帰時に安全に書き戻すフォールバック処理を実装しています。

## 4. 各コンポーネント主要仕様

### 4.1 ダッシュボード機能 (`ScenarioDashboard`)
- アプリ起動時のポータル画面として機能。
- 新規シナリオの作成 (直接タイトル入力可能) のほか、既存シナリオの一覧表示・ロード・削除機能を提供。

### 4.2 エディタ画面 (`TRPGEditor`)
本アプリケーションの中核となり、以下の主要サブコンポーネントで構成されます。

#### エディタ／プレビュー領域 (`EditorArea`)
- 左ペイン（テキスト入力）、右ペイン（Markdownリアルタイムプレビュー）の2画面構成。
- 長文編集時において、双方のスクロール位置が内容の高さの比率に応じて常に同期 (Sync) します。
- **VSCodeライクな検索・置換ウィジェット**を独自に備え、入力領域内の文字列を動的にハイライト・自動スクロールにて高速にナビゲーション・置換可能です。

#### ページ管理サイドバー (`PageSidebar`)
- シナリオ内の「ページ」要素を一覧で管理。
- **ドラッグ＆ドロップ機能**による柔軟な並べ替えに対応。
- **自動目次 (TOC) システム**: 選択中のページに含まれる大見出し(`#`)・中見出し(`##`) を解析し、ページ名から展開される「目次ナビゲーション」をリアルタイム生成。クリック時に該当行へエディタ・プレビュー共に自動スクロールします。

#### 登場人物・NPC管理サイドバー (`NpcSidebar`)
- シナリオに登場するNPC名の一元管理・リスト化機能。
- キャラクターの追加・削除の他、**リスト項目のクリックによるインラインリネーム対応**の実装。

#### 操作ツールバー (`EditorToolbar`)
- テキスト装飾、特定ブロック(情報枠や補足枠)の挿入、検索UIのトグル、画像追加ボタン等を格納。

#### キャラクターピッカー (`CharacterPicker`)
- `Ctrl+D` にて呼び出せる高速入力用モーダル。登録済みのNPC名から選択するだけで、即座にフォーマット済みのセリフ表記（例：`[NPC名] `）を本文へ挿入可能。

#### 設定・カスタマイズ機能 (`SettingsModal`)
- **エディタフォントの動的変更**: ゴシック(Noto Sans JP) / 明朝(Noto Serif JP) 等のフォントファミリー切替、およびpx単位のサイズ指定が可能。即時プレビューに反映されます。
- ダークモードとライトモードの自由な切替機能。
- アクションごとのショートカットキー割り当てカスタム機能の提供。

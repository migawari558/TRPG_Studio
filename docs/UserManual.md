# TRPG Studio 操作マニュアル

ようこそTRPG Studioへ！このアプリは、TRPGのシナリオ作成を驚くほど快適にするために作られた専用のテキストエディタです。

---

## 1. 画面の基本操作

### ダッシュボード
アプリを起動すると表示される「シナリオ管理画面」です。
- **＋ 新規作成**: クリックして名前を入力することで、新しいシナリオを作ります。
- **既存のシナリオ**: 過去に作ったシナリオをクリックして、すぐに執筆を再開できます。

### エディタ・プレビュー画面
エディタを開くと、画面が「左側（文章を書く場所）」と「右側（綺麗な完成図）」の二つに分かれています。
- 左側に文章を打ち込むと、リアルタイムで右側に美しいデザインが反映されます。
- 長い文章を書いても、左右のスクロールは常に連動して動くため、今どこを書いているか見失うことがありません。

---

## 2. シナリオ構成の管理（ページ機能）

シナリオの「チャプター」や「章」は、画面左のサイドバー（ページリスト）で管理します。
- **ページの追加**: ページリスト一番下の「＋ 新規ページを追加」をクリックします。
- **ページの削除**: ページ名の右端にある「ゴミ箱」アイコンで削除します。
- **並べ替え**: ページ名をマウスでドラッグ＆ドロップして、自由に順番を入れ替えられます。

**✨ 自動目次とジャンプ機能**
文章の中で見出し（行の先頭に `# ` や `## ` を付ける）を作ると、左サイドバーのページ名の横にある `>`（矢印アイコン）から**「自動目次」**を開くことができます。これを使うと、目次をクリックするだけでエディタが一瞬でその場所にジャンプするため、非常に便利です！

---

## 3. キャラクター（NPC）の管理

右側のサイドバー（人のアイコンのボタンで開閉）から、登場人物を登録できます。
- **キャラクターの追加**: 上部のテキストボックスに名前を入れて「＋」ボタンを押すだけです。
- **キャラクター名の変更**: リスト上のキャラクター名をクリックすると、その場で名前を直接書き換えることができます。（Enterキーで確定）
- **セリフの素早い挿入**: 執筆中に `Ctrl + D` （または `Command + D`）を押すと「キャラクターピッカー」が開き、選んだキャラクターのセリフとして本文にワンタッチで挿入できます。

---

## 4. 便利機能・ショートカットキー

執筆スピードを上げるための便利機能が揃っています。よく使うものはぜひショートカットキーで活用してください。（※キーは設定から自由に変更可能です）

| 機能名 | デフォルトキー | 説明 |
| :--- | :--- | :--- |
| **太字** | `Ctrl + B` | 選択した文字を強調（太字）にします |
| **見出し** | `Ctrl + Shift + H` | 見出し（ `## ` ）を挿入します |
| **セリフ追加** | `Ctrl + D` | キャラクター一覧から選んでセリフを挿入します |
| **情報ブロック**| `Ctrl + I` | コラム風の「情報枠ブロック」を挿入します |
| **検索** | `Ctrl + F` | エディタ内の文字を検索します |
| **置換** | `Ctrl + H` | エディタ内の文字を検索し、別の文字に一括で置き換えます |

### 🔍 検索・置換機能について
`Ctrl + F` を押すと、画面右上に半透明の「検索パネル」が現れます。
- ヒットした件数が表示され、エンターキーや上下の矢印ボタンで瞬時に該当箇所へジャンプします。
- パネルの左端にある `>` をクリックすると、その文字を別の文字に変える「置換用」の入力欄が展開されます。

---

## 5. エディタの設定変更

画面右上の「歯車アイコン」から設定画面を開けます。
- **フォントの種類・サイズ**: 執筆に集中できるよう、文字の大きさと種類（ゴシック体・明朝体など）を自由に変更できます。即座にエディタに反映・保存されます。
- **ダークモード・ライトモード切替**: 夜間の執筆など、好みに合わせて画面の色調（太陽／月のアイコン）を切り替えられます。
- **データの保存先**: シナリオデータの保存先フォルダを変更・確認できます。万が一USBメモリを抜いてしまった等のエラーが起きてもアプリ内部にバックアップがされるため安心です。
